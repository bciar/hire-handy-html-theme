# Gulp and plugins instalation. Compilation of SASS and Pug, Livereload, Images Optimization

1. Install Node.js
2. Open terminal (or command prompt in Windows) and navigate to project folder where files "gulpfile.js" and 
"package.json" are located.
3. In terminal run "npm install".
4. In terminal run 'gulp'. It will start server, SASS/Pug compilation, Livereload.
5. Run production task - "gulp prod" - cleaning files, optimizes images, minify CSS etc.
